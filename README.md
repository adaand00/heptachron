### Features
 - [x] 4 digits (hours and minutes)
 - [x] Seconds displayed in binary (6 bits)
 - [x] Always-on display
 - [x] Rechargeable battery
 - [x] Magnetic charger
 - [ ] Long battery life
 - [x] Megatron
 - [x] Not waterproof
 - [ ] Very accurate
