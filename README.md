### Features
 - [ ] 4 digits (hours and minutes)
 - [ ] Seconds displayed in binary (5 bits)
 - [ ] Always-on display
 - [ ] Rechargeable battery
 - [ ] Magnetic charger
 - [ ] Long battery life
 - [x] Megatron
 - [ ] Not waterproof
 - [ ] Very accurate
